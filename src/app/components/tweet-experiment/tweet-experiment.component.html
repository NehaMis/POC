📋 Findings Summary
❌
Metadata fetch (frontend-only)
NOT POSSIBLE. oEmbed is CORS-blocked in browsers. X API v2 requires a server-side Bearer Token. Both need a backend proxy.
✅
Targeted reply (frontend-only)
POSSIBLE. in_reply_to=<tweetId> in the Intent URL is respected by X. The original tweet appears above the composer. No auth required on our side.
